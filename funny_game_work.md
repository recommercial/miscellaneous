# 故事大綱
某天公司新來個帥同事,不久所有女同事都跟帥同事越走越近,
連玩家(Player)心儀已久的女同事跟帥同事關係也越發密切,快被搶走了.
在偶然的情況下,發現了帥同事頭髮有點歪歪的,萌發了想讓帥同事假髮掉下來
的報復思想,並搶回女友
